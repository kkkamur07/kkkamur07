## What I'm working on

1. **Active learning** — With Prof. Schetze: comparing uncertainty measures to choose training data so models learn more from less.
2. **Indic encoders** — Training ModernBERT-style models for **Indian languages**. Distilled Hindi checkpoints are [on Hugging Face](https://huggingface.co/kkkamur07); I'm extending the line and aiming to finish by June.
3. **Greek Foundation** — An end-to-end ML pipeline to transcribe **Byzantine Greek** manuscripts—segmentation, text extraction, and deployment—along the lines of eScriptorium for LMU.
4. **Needle in a haystack** — Stronger RAG when the answer sits mid-document, where positional bias in embeddings hurts retrieval.
