## What I'm working on

1. **Active learning** — Project with Prof. Schetze (pioneer of information retrieval): we're evaluating uncertainty measures to pick the right data for training models, aiming for better performance with smaller datasets.
2. **Indic encoders** — Training modern BERT encoders for Indian languages. Distilled Hindi models are [on Hugging Face](https://huggingface.co/kkkamur07); I'm extending the project and aiming to wrap up by June.
3. **Greek Foundation** — An ML system that automatically transcribes Byzantine Greek manuscripts: segmentation, text extraction, and a proper production setup for users and model deployment—kind of like eScriptorium for LMU.
4. **Needle in a haystack** — Improving retrieval for needle-in-haystack setups in RAG. Positional embedding bias means retrieval works well for text at the start and end, but not in the middle—working on that.
